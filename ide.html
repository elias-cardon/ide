Différents IDE



Sublime Text 2

Sublime Text est un éditeur de texte vraiment puissant.

Il est conçu pour prendre en charge plusieurs langages de programmation variés allant du langage de programmation C à l'Action Script en passant par les langages PHP, Objective-C ou encore OCaml voire même du Scripting comme le Shell Scripting ou encore le SQL.

Visual Studio Code

C'est un éditeur de code open-source, gratuit et multi-plateforme (Windows, Mac et Linux), développé par Microsoft, à ne pas confondre avec Visual Studio, l'IDE propriétaire de Microsoft.

Principalement conçu pour le développement d'application avec JavaScript, TypeScript et Node.js, l'éditeur peut s'adapter à d'autres types de langages grâce à un système d'extension bien fourni.

Notepad++

Notepad ++ est un éditeur de texte comme le Bloc-Notes (Notepad de Windows).

Mais Notepad++ sait aussi éditer d’autres fichiers tels que les fichiers html, php, et bien d’autres types de fichiers.
